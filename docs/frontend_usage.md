# 前端界面使用说明（Flask）

## 功能

- 上传本地图片
- 选择任务：`low_light` / `denoise` / `super_resolution` / `inpainting` / `face_restoration`
- 使用对应任务权重进行推理并显示结果图
- 图像补全支持画笔涂抹掩膜（推荐）

## 图像补全画笔说明

- 选择任务为 `inpainting` 后，可在“补全画笔”区域直接涂抹需要消除/补全的位置
- 白色涂抹区域表示待补全区域，支持橡皮擦和清空
- 也可上传掩膜文件作为替代
- 若同时存在“画笔掩膜”和“上传掩膜文件”，系统优先使用画笔掩膜

## 图像补全最小可用优化包（已启用）

- 自动缺失区域识别：无手动掩膜时，系统优先自动检测疑似缺失区域
- 软掩膜融合：对掩膜边界进行羽化，减轻边界割裂感
- 无缝融合后处理：通过无缝克隆降低修补区域与周围背景的边界感

推荐使用流程：

1. 开启“自动识别缺失区域”得到初始修补结果
2. 若区域不准确，使用画笔微调掩膜后再次处理

## 启动

```bash
python app.py
```

默认地址：`http://127.0.0.1:7860`

## 权重规则

- 默认自动读取：`experiments/checkpoints/best_<task>.pth`
- 若权重在其他路径，可在界面中填写“权重路径（可选）”

## 常见问题

1. 报错“未找到权重文件”
   - 先执行对应任务训练，生成权重文件。

2. 打不开页面
   - 检查终端是否输出 `Running on http://127.0.0.1:7860`，并确认 `7860` 端口未被占用。
